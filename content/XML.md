**O que é um arquivo XML?**

Um arquivo .xml (Extensible Markup Language) é um tipo de arquivo de dados que utiliza uma linguagem de marcação para armazenar informações em formato estruturado. Isso significa que o conteúdo do arquivo está organizado em tags, chamadas "etiquetas", que definem a natureza e a hierarquia dos dados.

**Características principais**

* É uma linguagem de marcação aberta e flexível, permitindo a definição de novas tags e estruturas personalizadas.
* Utiliza um padrão de sintaxe específico para criar arquivos que podem ser facilmente lidos por máquinas e humanos.
* Pode ser utilizado em uma ampla variedade de aplicações, desde armazenamento de [[dados]] até comunicação entre sistemas.

**Exemplo básico**

[[Estrutura]] Arquivo XML:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<livro>
  <titulo>Meu Primeiro Livro</titulo>
  <autor>Jane Doe</autor>
  <ano-publicacao>2022</ano-publicacao>
</livro>
```

**Tags e suas propriedades**

* A tag `<xml version="1.0" encoding="UTF-8"` é uma declaração de versão do arquivo XML.
* As tags `<livro>` e `</livro>` definem o nível superior da estrutura, que contém informações sobre um livro específico.
* As tags `<titulo>`, `<autor>` e `<ano-publicacao>` são sub-tags que fornecem mais detalhes sobre o livro.

**Uso comum**

Arquivos XML são frequentemente utilizados em aplicações como:

* Armazenamento de dados para bancos de dados ou aplicativos
* Comunicação entre sistemas, permitindo a troca de informações entre diferentes softwares e plataformas
* Geração de conteúdo dinâmico, como páginas web ou relatórios
* Processamento de dados em batch, como importação/exportação de arquivos
